## Installing

Using npm:

```bash
$ npm install @agility/plenum-ui
```

Using bower:

```bash
$ bower install @agility/plenum-ui
```

Using yarn:

```bash
$ yarn add @agility/plenum-ui
```


## Usage

```js
import { Button } from '@agility/plenum-ui';

<Button label='Primary' type='primary' size='base' icon='BellIcon'>
```
